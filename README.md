# Sloth Weekly Challenge

A collection of JavaScript coding challenges, including algorithm problems and exercises.

## Contents

- `challenge_January-7th,2026.js` - Weekly challenge: Remove last vowel from words in sentences

## Challenge Description

### Remove Last Vowel
This challenge involves removing the last vowel from each word in a given sentence while preserving punctuation.

**Example:**
```javascript
removeLastVowel("Those who dare to fail miserably can achieve greatly.")
// Output: "Thos wh dar t fal miserbly cn achiev gretly."
```

## Getting Started

To run the challenges:

```bash
node challenge_January-7th,2026.js
```
